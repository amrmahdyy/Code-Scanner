- UIKit
- Vision
- AVFoundation



## Requirements

iOS 15.0+


# Features

- Supporting codes such as (qr, codabar, aztec, upce)
- Camera Flash Mode
- Persistent storage
- touch & hold gesture/3D Touch
- Haptic feedback
